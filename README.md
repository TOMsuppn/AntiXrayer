# AntiXrayer 反矿透
![p1](https://cdn.modrinth.com/data/cached_images/01fab1b1e2b61f12627eef71417abf95bd7b0953_0.webp)
[Antixrayer 宣传视频(点击查看)](https://www.bilibili.com/video/BV1fyedzbE4x)

当玩家挖掘自然生成的钻石，远古残骸时 OP以及拥有权限的管理员可收到消息提醒并可隐身传送至可疑玩家附近


When players mine naturally generated diamonds and ancient debris, OPs and administrators with permissions will receive notifications and can become invisible near suspicious players according to legend. If a player's behavior is suspicious, this plugin will automatically freeze the suspicious player's account and require them to take a screenshot of the current game screen for administrator review.

## Commands 命令

```
/XrayerDetector <Player> Ban unverified miners from accessing the game.临时封禁可疑玩家

/axunban <Player> unban xrayer player.解封玩家

/axv Switch vanish mode.切换隐身模式

/axb Back to last tp location.返回上次传送点

/axview <Player> Enable review mode 启动审查模式

```

## Permissions 权限

```
antixrayer.xrayerdetector 使用/XrayerDetector <Player>
antixrayer.unban 使用/axunban <Player>
antixrayer.vanish 使用/axv
antixrayer.back 使用/axb
antixrayer.view 使用/axview
antixrayer.bypass 绕过反矿透检测
antixrayer.alarm 接受反矿透报警
```
